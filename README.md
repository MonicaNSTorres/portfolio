# 🚀 Mônica Torres Portfolio

Portfólio profissional desenvolvido com *Next.js, **TypeScript* e *Tailwind CSS*, com foco em apresentar projetos, experiência full stack e soluções web modernas.

Este site foi criado para demonstrar minha experiência como *Desenvolvedora Full Stack*, destacando projetos reais, arquitetura moderna e soluções voltadas para problemas de negócio.

# ▶ Como rodar o projeto localmente

Clone o repositório:


git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git


Entre na pasta do projeto:


cd SEU-REPOSITORIO


Instale as dependências:


npm install


Execute o ambiente de desenvolvimento:


npm run dev


A aplicação estará disponível em:


http://localhost:3000


---

# 🔐 Variáveis de ambiente

Crie um arquivo .env.local na raiz do projeto.

Exemplo:


RESEND_API_KEY=SUA_CHAVE_AQUI
CONTACT_TO_EMAIL=seuemail@dominio.com


Essas variáveis são utilizadas para envio de mensagens do formulário de contato.
